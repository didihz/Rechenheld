🔄 Dezimalzahl ↔ Bruch
1
Dezimalzahl → Bruch: Nachkommastellen zählen = Anzahl Nullen im Nenner
0,23 = 23/100    0,7 = 7/10    0,003 = 3/1000
2
Bruch → Dezimalzahl: Nenner bestimmt die Stelle
7/10 = 0,7    15/1000 = 0,015    89/100 = 0,89
⚖️ Dezimalzahlen vergleichen

Die Strategie

1. Stelle für Stelle von links vergleichen
2. Erste unterschiedliche Stelle entscheidet
3. Nullen hinten anhängen macht den Vergleich leichter:
0,99 = 0,990 → gleich!     6,7 vs 6,666 → 6,7 > 6,666
💡 Achtung: 0,5 > 0,499 – nicht „mehr Ziffern = mehr Wert"!
🎯 Runden
1
Auf Einer runden: Erste Nachkommastelle anschauen. ≥5 → aufrunden, <5 → abrunden
1,7 ≈ 2    13,92 ≈ 14    25,46 ≈ 25
2
Auf Zehntel runden: Zweite Nachkommastelle anschauen
15,64 ≈ 15,6    3,88 ≈ 3,9    12,956 ≈ 13,0
3
Auf Hundertstel runden: Dritte Nachkommastelle anschauen
2,466 ≈ 2,47    13,8331 ≈ 13,83    7,209 ≈ 7,21

✖️ Komma bei Multiplikation
Regel: Alle Nachkommastellen beider Faktoren zusammenzählen = so viele Nachkommastellen hat das Ergebnis.

41,57 × 3 → 2 NKS + 0 NKS = 2 NKS → 124,71
8,4 × 7,3 → 1 NKS + 1 NKS = 2 NKS → 61,32
89,4 × 0,4 → 1 NKS + 1 NKS = 2 NKS → 35,76
