Was ist das?

Was ist eine Fläche?

Eine Fläche ist der Bereich innerhalb einer Form. Der Flächeninhalt sagt, wie viel Platz diese Form einnimmt.

🌱 Beispiel aus dem Alltag: Ein Garten, ein Zimmer, ein Spielfeld – all das sind Flächen. Den Flächeninhalt braucht man z. B., um zu wissen, wie viel Rasen man kaufen muss.
Schritt für Schritt

Grundformen erkennen

Fast alle Flächen bestehen aus einfachen Grundformen. Diese solltest du gut kennen:

Rechteck

A = a · b
Länge mal Breite

Quadrat

A = a · a = a²
Seite mal Seite

Dreieck

A = (g · h) / 2
Grundlinie mal Höhe, geteilt durch 2

Trapez

A = (a + c) / 2 · h
Mittlere Breite mal Höhe
Der Kern

Was sind zusammengesetzte Flächen?

Manche Flächen sehen kompliziert aus – aber man kann sie in einfache Formen aufteilen.

1
Teile auf: Erkenne, welche einfachen Formen darin stecken (z. B. 2 Rechtecke).
2
Berechne: Berechne den Flächeninhalt jeder Teilfläche einzeln.
3
Addiere: Addiere alle Teilflächen → das ist die Gesamtfläche.
💡 Tipp: Manchmal kann man auch die große Fläche nehmen und ein Stück abziehen:
A = Agesamt – Aentfernt
Beispiel komplett

Durchgerechnetes Beispiel

Ein L-förmiges Grundstück mit folgenden Maßen (in m):

1
A₁ (grünes Rechteck):
A₁ = 4 m · 8 m = 32 m²
2
A₂ (gelbes Rechteck):
A₂ = 8 m · 3,5 m = 28 m²
3
Gesamt:
A = A₁ + A₂ = 32 + 28 = 60 m²
Im Alltag

Anwendung: Rasenproblem

Wenn du die Fläche kennst, kannst du damit weiterrechnen:

🌿 Aufgabe: Pro m² Rasenfläche braucht man 2,4 kg Rasensamen. Wie viel braucht man für 60 m²?
1
Fläche berechnen → A = 60 m² (aus dem Beispiel oben)
2
Rasenmenge = 60 m² · 2,4 kg/m² = 144 kg
🔢 Einheiten nicht vergessen! Fläche → m²  |  Länge → m  |  Gewicht → kg

Flächenformeln

Wichtige Formeln

Rechteck
A = a · b    |    U = 2 · (a + b)
a = Länge, b = Breite
Quadrat
A = a²    |    U = 4 · a
a = Seitenlänge
Dreieck
A = (g · h) / 2    |    U = a + b + c
g = Grundlinie, h = Höhe
Trapez
A = (a + c) / 2 · h
a, c = parallele Seiten, h = Höhe
Zusammengesetzte Flächen
Agesamt = A₁ + A₂ + A₃ + ...
Oder: Agesamt = Agroß – Aentfernt

Strategie

Wie zerlege ich eine Fläche?

Es gibt zwei Strategien:

Addieren ➕

Subtrahieren ➖
Große Fläche berechnen und Ausschnitte abziehen.
